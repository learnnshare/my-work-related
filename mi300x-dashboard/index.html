About this mockup. Built as a self-contained set of HTML/CSS/JS files (no internet required — open index.html directly). The knobs on each dashboard drive a shared simulation engine (assets/sim.js) that produces plausible-but-fake metrics reflecting real behaviour: batch-1 latency is launch-bound, decode is bandwidth-bound, multi-GPU scaling loses efficiency, and so on. Replace the sim engine with real telemetry to turn this into the production console.
